Instalasi Database Baru

Untuk membuat database baru bagi cabang/toko ini, ikuti langkah berikut:

  1. Klik tombol Unduh Kode GS di bawah ini.
  2. Buka Google Sheets kosong baru.
  3. Pilih menu Ekstensi > Apps Script.
  4. Buka file yang baru diunduh, lalu Copy-Paste (Salin-Tempel) semua kodenya ke editor Apps Script.
  5. Klik ikon Simpan (Save).
  6. Klik tombol biru Terapkan (Deploy) > Deployment Baru.
  7. Pilih jenis Aplikasi Web (Web App), atur akses ke Siapa saja (Anyone), lalu klik Terapkan.
  8. Salin URL Web App yang muncul dan tempelkan ke kolom Tautan di kotak paling atas.
